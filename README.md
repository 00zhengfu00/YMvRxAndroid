**因公司项目重构，故此搭建一套属于自己的基础库，方便以后开发升级使用，采用Mvrx+Retrofit2(okhttp3)+rxjava2+kodein+objectBox+Glide+rxlifecycle。搭建一套整体架构**

#### 主要技术
+ Kotlin
+ Retrofit + Okhttp
+ RxJava + RxAndroid
+ liveDatabus
+ kodein
+ MvRx
+ Glide
+ SmartRefreshLayout
+ objectBox
+ lottie
